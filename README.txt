!!! PENTRU A TESTA TOATE FACILITATILE WEBSITE-ULUI !!!

Puteti folosi una din urmatoarele metode:

1) Accesati Link-ul https://amarkayedfmi.github.io/ProiectTW/

2) Descarcati folder-ul cu proiectul de pe https://github.com/AmarKayedFMI/ProiectTW
    Odata in folder-ul cu proiectul descarcat, 
    deschideti command prompt pentru acest folder 
    tastati comanda "http-server -c-1"
    In sectiunea "Available on:" o sa aveti 2 link-uri
    Copiati unul/oricare din link-urile respective si accesati-le intr-un browser(Chrome, Mozilla Firefox, MS Edge, etc)

3) Instalati Visual Studio Code si extensia "Live Server" din cadrul acestui program
    descarcati folder-ul cu proiectul
    deschideti folder-ul in in acest program si deschideti fisierul "index.html"
    in acest fisier faceti click dreapta si selectati "Open with Live Server"
    sau tineti butonul "Alt" apasat si tastati(in aceasta ordine) "L" si "O" pentru a realiza combinatia "Alt+L Alt+O"
    
Puteti folosi oricare din metodele de mai sus, nu trebuiesc aplicate toate simultan.
Nu este suficient sa deschideti doar fisierul "index.html" in browser, intrucat nu se va realiza conexiunea cu server-ul si nu se vor putea testa decat facilitatile de front-end.
